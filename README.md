# AI Tutor System 🤖📚
This is an AI-powered tutoring system built using Python, FastAPI, Gradio, and Ollama (TinyLlama).  
It allows students to interact with an AI model for learning support, explanations, and academic assistance.

Features
- AI chatbot for student learning support
- Powered by TinyLlama via Ollama
- FastAPI backend for handling requests
- Gradio UI for easy interaction
- Modular and expandable structure

#Technologies Used
- Python
- FastAPI
- Uvicorn
- Gradio
- Ollama (TinyLlama)
- Requests
- Pydantic
- Python-dotenv
